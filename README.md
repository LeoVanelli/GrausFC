## Exercício `GrausFC`

Converta uma temperatura digitada pelo usuário em °F para °C.

°C = (°F − 32) / 1,8

Ref.: [https://pt.wikipedia.org/wiki/Celsius](https://pt.wikipedia.org/wiki/Celsius)

```
°F = 0
0°F equivalem a -17,78°F
```
